## 🧩 Problem Statements / Questions Solved

This project is structured around solving real-world data analysis questions using Python and Pandas.

### 📊 Dataset Exploration

1. Display top 10 rows of the dataset
2. Display last 10 rows of the dataset
3. Check data type of each column
4. Identify null values in the dataset
5. Determine total number of rows and columns

---

### 💰 Purchase Analysis

6. Find the highest and lowest purchase prices
7. Calculate the average purchase price

---

### 🌍 User Behavior Analysis

8. Count users with French (`fr`) as their language
9. Identify job titles containing "Engineer"

---

### 🔍 Targeted Queries

10. Find the email of the user with IP address **132.207.160.22**
11. Count users who use **Mastercard** and made purchases above **50**
12. Find the email of the user with credit card number **4664825258997302**

---

### ⏰ Time-Based Analysis

13. Analyze how many purchases were made during **AM vs PM**

---

### 💳 Credit Card Insights

14. Count users whose credit card expires in **2020**

---

### 📧 Email Analysis

15. Identify the **top 5 most popular email providers**

---

## 🎯 Learning Outcome

Through these problems, the project demonstrates:

* Practical usage of **Pandas for real-world analysis**
* Data filtering, aggregation, and querying techniques
* Extracting meaningful insights from structured data
* Translating business questions into data-driven answers
