# 📊 Data Analysis with Python – Employee Dataset

This project focuses on performing **Explatory Data Analysis (EDA)** on an employee dataset using Python libraries like **Pandas** and **NumPy**.

The goal is to understand the dataset step by step by solving structured questions and extracting meaningful insights.

---

## 🚀 Project Overview

In this project, we solve **16 real-world data analysis questions** to explore, clean, and analyze the dataset effectively.

---

## 📌 Questions Covered

1. Display Top 10 Rows of The Dataset  
2. Check Last 10 Rows of The Dataset  
3. Find Shape of Dataset (Rows & Columns)  
4. Get Dataset Info (rows, columns, datatypes, memory usage)  
5. Check Null Values  
6. Drop unnecessary columns (`Id`, `Notes`, `Agency`, `Status`)  
7. Get Overall Statistics of DataFrame  
8. Find Occurrence of Employee Names (Top 5)  
9. Find Number of Unique Job Titles  
10. Count Job Titles Containing "Captain"  
11. Display Employees from Fire Department  
12. Find Min, Max, and Average BasePay  
13. Replace `'Not Provided'` with `NaN`  
14. Drop Rows Having 5 Missing Values  
15. Find Job Title of **ALBERT PARDINI**  
16. Find Total Earnings of **ALBERT PARDINI (Including Benefits)**
