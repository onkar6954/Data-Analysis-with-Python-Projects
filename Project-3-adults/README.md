# 📊 Data Analysis with Python – Data Cleaning & Sampling

This project focuses on **data preprocessing and exploration techniques** using Python. It covers essential operations like handling missing values, removing duplicates, filtering data, and working with data types.

The goal is to build a strong foundation in **data cleaning and manipulation using Pandas**.

---

## 🚀 Project Overview

In this project, we explore important data analysis concepts and apply them through practical questions.

---

## 📚 Concepts Covered

- 🎯 Fetching Random Samples from Dataset  
- 🔍 `isin()` – Filtering specific values  
- 📏 `between()` – Range-based filtering  
- 🔑 `unique()` – Finding unique values  
- 🧹 `dropna()` – Handling missing data  
- 🔁 `replace()` – Replacing values  
- 🔍 `duplicated()` – Detecting duplicates  
- ❌ `drop_duplicates()` – Removing duplicates  
- 🔄 `astype()` – Changing data types  

---

## 📌 Questions Solved

1. Display Top 10 Rows of The Dataset  
2. Check Last 10 Rows of The Dataset  
3. Find Shape of Dataset (Rows & Columns)  
4. Get Dataset Info (rows, columns, datatypes, memory usage)  
5. Fetch Random Sample (50% of dataset)  
6. Check Null Values  
7. Perform Data Cleaning (Replace `'?'` with `NaN`)  
8. Drop All Missing Values  
9. Check and Remove Duplicate Records
